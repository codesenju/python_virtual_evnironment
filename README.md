# Creating a python virtual evnironment
### Install
```pip3 install virtualenv```
### Now we can create a new python virtual environment:
```mkdir -p ~/.virtualenvs``` \
```cd ~/.virtualenvs``` \
```virtualenv dev-env``` \
```source dev-env/bin/activate``` \
```pip3 -V``` \
```pip3 list``` \
```pip3 install paramiko``` \
```pip3 list```
### Exit virtual environment:
```deactivate```
